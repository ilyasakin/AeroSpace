:manpurpose: Center the window on its monitor (floats tiled windows first)

// =========================================================== Synopsis
== Synopsis
[verse]
// tag::synopsis[]
aerospace center-window [-h|--help] [--window-id <window-id>]

// end::synopsis[]

// =========================================================== Description
== Description

// tag::body[]
{manpurpose}

The window keeps its size and is positioned in the center of the monitor's visible area
(respecting outer gaps).

If the window is tiled, it is turned into a floating window first, the same way
xref:aerospace-layout.adoc[layout floating] does it.
